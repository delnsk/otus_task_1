## Текст промпта

```
сейчас вопросы вываливаются единым списком, нужно изменить логику - задавать вопросы по очереди и отображать прогресс сколько отвечено и сколько осталось (можно сегментным индикатором)
```

## Результат

```
Логика анкеты переведена на пошаговый режим.

Что изменилось
Один вопрос за раз — показывается только текущий вопрос (QuestionStep), навигация кнопками «Назад» / «Далее», на последнем шаге — «Отправить ответы».

Прогресс (SurveyProgress):

счётчики «Отвечено: X» и «Осталось: Y»;
сегментная полоса (по одному сегменту на вопрос): заполненные — с ответом, текущий — с обводкой;
подпись «Вопрос N из M».
Валидация по шагам — обязательный вопрос проверяется при переходе «Далее» и при отправке; при вводе ответа ошибка сбрасывается.

Удалён QuestionList.tsx — вместо списка всех карточек используется пошаговый компонент.

Сборка фронтенда проходит успешно. Запустите npm run dev в корне проекта, чтобы проверить в браузере.
```
