## Текст промпта

```
Проанализируй требования в @docs/source_task.md

Дополнительные требования к реализации backend:
- сервис на NodeJs
- код должен быть написан с соблюдением принципов SOLID
- вопросы хранятся в отдельном файле, в json формате. Вопросы могут быть как открытые, так и с вариантами ответа. Открытый вопрос - текстовое поле поле, вопрос с одним валидным ответом - переключатель, вопрос с несколькими валидными ответами - чекбоксы.
- ответы сохраняются в паке answers в формате json (для каждого ответа гененрируется новый файл с индексом). По мимо ответов должны сохраняться вопросы

Дополнительные требования к реализации frontend:
- реализация на React
- код должен быть написан с соблюдением принципов SOLID 
- используй библиотеку shadecn

Дополнительные требования к UI/UX:
- используй минималистичные современные стили, добавть не много не новящевой анимации, цветовая схема должна быть тонах синего

Дополнительные требования к деплою:
- заполни файл с вопросами (что бы были вопросы всех трех типов) на тему "Приведет ли развитие ИИ к вымиранию профессии программирования"
- создай скрипт по устаноке всех зависимостей и запуску сервиса и фронтенд части
- создай в папке docs мркдаун документ с кратким описание работы приложения и инструкцию по его запуску

На основе вышеописанных требований создай детальный план для реализации и сохрани его в формате md в папку docs, если требуются какие то уточнения- задай дополнительные вопросы
```
